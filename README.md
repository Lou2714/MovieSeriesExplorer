# Movies & Series Explorer

Aplicación web para explorar películas y series utilizando la API de TMDB. 

## Descripción
Movies & Series Explorer es una aplicación web construida con React que permite explorar series y películas. Los usuarios pueden buscar títulos, filtrar por género y guardar películas o series en favoritos.

## Funcionalidades
- Descubrir las películas mejor puntuadas
- Descubrir las series más populares
- Buscar películas y series
- Filtrar por género
- Ver detalles de cada película o serie
- Añadir contenido a favoritos
- Navegación entre páginas
- Diseño responsive

## Instalación
1. Clonar el repositorio
```
https://github.com/Lou2714/MovieSeriesExplorer.git
```
2. Entrar en la carpeta del proyecto
```
cd movieSerieExplorer
```
3. Instalar dependencias
```
npm install
```
## Variables de entorno
Este proyecto utiliza la API de TMDB, por lo que es necesario crear un
archivo `.env` en la raíz del proyecto.

Ejemplo:

VITE_TMDB_API_KEY=su_api_key

Para obtener una API key de TMDB se debe crear una cuenta y acceder a las configuraciones de su perfil para solicitarla.

[Enlace para crear una cuenta](https://www.themoviedb.org/signup)

[Enlace para solicitar la API key](https://www.themoviedb.org/settings/api)

## Ejecutar el proyecto
```
npm run dev
```
